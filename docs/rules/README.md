Rules Documents
for Cloud-Native Component Framework

This directory contains supplementary rule documents.

The authoritative, top-level rule document is `RULE.md`
at the repository root.

Documents under docs/rules/ provide supporting, detailed,
or topic-specific rules and must not contradict RULE.md.

Relationship to core rules
--------------------------
Core defines primitives and base semantics.
In CNCF, these rules contextualize core usage and define
composition and execution structure.
